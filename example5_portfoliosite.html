Graphic / Web Design - Photo Editing

For my site I wanted a simplistic site layout that would get the end user what they wanted to find quickly. I did this by centering my elements as many people look to the center of their screen first. I added a footer aswell to make information easier to find. For this site I coded it entirely in HTML and CSS with a little React and java. My favorite element of the site is the slideshow on the home page. There is a slight parallax effect on hover which makes the images pop out more. One change that I will make is to make the page more mobile friendly, most of the elements work rather well but some don't scale correctly which is something I need to fix.
